### How to Start a New Research Project

1. Launch Autonomous Multi-Branch Research Lab
2. Click **"New Project"**
3. Enter research topic (e.g., "Optimizing transformer models for low-resource languages")
4. Select number of branches (1-10, default 5)
5. Choose literature sources (ArXiv, PubMed, both)
6. Click **"Initialize Branches"**
7. AI begins parallel research
8. Monitor progress in dashboard

### How to Review Literature Automatically

1. In project dashboard, click **"Literature"** tab
2. AI shows discovered papers (10-200 depending on topic)
3. Each paper shows:
   - Title, authors, year, venue
   - AI-generated summary (1-2 paragraphs)
   - Relevance score (0-100%)
   - Citation count
4. Click paper to read full abstract
5. Check papers to include in review
6. Click **"Generate Literature Review"** → AI writes section

### How to Generate and Evaluate Hypotheses

1. Click **"Hypotheses"** tab
2. AI shows 5-20 generated hypotheses
3. Each hypothesis includes:
   - Statement (testable claim)
   - Novelty score (0-100%)
   - Plausibility score (0-100%)
   - Falsifiability (easy/hard)
   - Suggested experiment design
4. Click **"Accept"** to save promising hypotheses
5. Click **"Explore Branch"** to spawn new branch for a hypothesis

### How to Analyze Data

1. Click **"Analysis"** tab
2. Upload data (CSV, Excel, JSON, or generate synthetic)
3. AI auto-detects:
   - Variable types (continuous, categorical)
   - Missing values
   - Outliers
4. Select analysis type or let AI recommend:
   - A/B test comparison
   - Regression modeling
   - Correlation analysis
   - Machine learning prediction
5. Click **"Run Analysis"**
6. View results with visualizations
7. Export analysis as LaTeX-ready table

### How to Generate a Paper

1. After research complete, click **"Write Paper"** tab
2. Select writing style:
   - **Conference Paper** (6-8 pages)
   - **Journal Article** (10-15 pages)
   - **Brief Report** (2-4 pages)
   - **Review Paper** (15-20 pages)
3. Choose citation format (APA, MLA, IEEE, Chicago)
4. Click **"Generate Draft"**
5. AI writes paper with:
   - Abstract
   - Introduction
   - Methods
   - Results
   - Discussion
   - References (auto-populated)
6. Edit manually or regenerate sections
7. Click **"Export"** → PDF/LaTeX/DOCX

### How to Merge Successful Branches

1. In project dashboard, click **"Branches"** tab
2. Review each branch's progress summary
3. Check branches with highest success scores
4. Click **"Merge Selected"**
5. AI combines insights from multiple branches
6. Merged outputs include:
   - Consolidated literature review
   - Unified hypothesis set
   - Combined experiment recommendations
   - Integrated manuscript draft
